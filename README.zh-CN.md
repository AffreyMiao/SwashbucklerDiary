# SwashbucklerDiary(侠客日记)

[English](./README.md) | 简体中文

「SwashbucklerDiary」是一个开源跨平台的、专注于日记的应用。

<img src="./Images/Windows.jpeg" />

<img src="./Images/Android.jpeg" />

## 支持的平台
- Windows (Windows 10/11)
- Android (Android 7.0以上)
- iOS (iOS 14以上)
- macOS (macOS 11以上)
